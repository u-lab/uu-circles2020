## 環境

- Vue.js
- Nuxt.js v2.12.0
- Firebase

## 環境構築

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

```

## デプロイ

このサイトはFirebaseでホスティングしています。
